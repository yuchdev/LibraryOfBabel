# Model Chain Reference

This directory documents the article-aligned model chain implemented in `src/babel/generators/`.

- Stage 0: `borges-library`
- Stage 1: `word-based`
- Stage 2: `punctuation-constrained`
- Stage 3: `sentence-structured`
- Stage 4: `grammar-constrained`
- Stage 5: `semantic-constrained`
- Stage 6: `topic-coherent`
- Stage 7-8: theoretical only (not application modes)
